耐力訓練是通過控制射精和高潮來增加克隆人耐力的一種訓練方式，也是對戰士質量進行評級的手段。
通常訓練時長為30分鐘，1小時，2小時，半天，全天，一周。每當戰士能夠在規定時間內不射，便能進入下一個時間更長的訓練任務，並得到對應的"榮譽分數" 。反之若隨意射精，將會受到懲罰，重新開始訓練週期，一個月禁止射精並減低分數。
榮譽分數越高的個體便能加入更加精英的戰隊，反之，沒有價值的個體將會被剝奪生育權，會遭到閹割並將切下的生殖器放到市場出售，其本身也會淪為商品，或成為有錢人的洩欲工具。

這是一段佔滿全屏的文字

给后来画手的一条小心得，也是是写给所有挣扎在数字里的创作者，作为前辈应该给后辈一些帮助：
粉丝数并不是衡量创作者价值的唯一标准，也不是决定你的生活是否成功和幸福的关键。
任何创作者的作品都有其独到的见解，都有其独特的价值，数字最多也只是市场的表现形式而已，你画的快，有一定基础，主题或角色吸引力强，就会有人看。但请千万别把数字和自己作品的价值挂钩，那更不是你人生的价值。在我心目中每一个画手都很棒，都可以成为我的老师，任何有独到见解，所有在努力的人都值得尊重。
一个人可以喜欢很多不同的创作者，并不会因为喜欢一个人而讨厌另一个人，同样你也无法控制别人的喜好。 粉丝数比别人少，并不代表画师的水平比别人差。粉丝数比别人多，也不代表你的画技和能力就比别人好。如果你十分幸运的得到了市场的宠幸，那么恭喜你，你将取得商业上的成功，迎合了大众的口味。
如果相反，也别自责，既不会让你的生活变得悲惨，也不代表你没有价值，只是市场规律而已，别让他毁掉了你的生活和你美丽的头脑。当然如果你想取得商业上的成功的话，不妨试试看在市场和自我表达上寻找一个平衡点。只要你能表达自己的想法，不论是取得商业还是艺术上的成就，或仅仅是为了开心都值得称赞
不管你怎么做，我想让你们明白，只有你的思想，你的心理健康，在创作中感受到的快乐和创造自己世界的成就感才是最重要的。别人创作什么，得到了什么，跟你没有任何关系。你可以从对方身上学习，甚至成为朋友。这个世界上并不是只存在竞争的二元对立关系，人是一种十分复杂的生物，也是自私的生物。但正是因为我们懂得合作，懂得欣赏彼此的差异，分享彼此的情感，才是人类的闪光点。如果你对竞争感到疲惫，不妨问问自己：究竟我想成为什么样的人，想做什么事情，别人所追求的成功真的是我想要的成功吗，别人所追求的成功是成功的唯一可能性吗？是否还有其它选择和路径呢？
就像海明威说过的：真正的高贵不是比别人优越，而是超越过去的自己 。
最后：
希望所有朋友能够专注于自己的成长和提升，自己的情绪快乐和健康，同时请记住：兽圈也不是你人生的全部，那只是你兴趣生活中的一部分而已，请多多照顾身边那些在意你的人和你在意的人。
