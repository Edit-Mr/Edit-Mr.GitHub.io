差分

耐力训练是通过控制射精和高潮来增加克隆人耐力的一种训练方式，也是对战士质量进行评级的手段。通常训练时长为30分钟，1小时，2小时，半天，全天，一周。每当战士能够在规定时间内不射，便能进入下一个时间更长的训练任务，并得到对应的"荣誉分数" 。反之若随意射精，将会受到惩罚，重新开始训练周期，一个月禁止射精并减低分数。荣誉分数越高的个体便能加入更加精英的战队，反之，没有价值的个体将会被剥夺生育权，会遭到阉割并将切下的生殖器放到市场出售，其本身也会沦为商品，或成为有钱人的泄欲工具。
